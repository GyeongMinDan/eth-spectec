.. _syntax-expression:

Expressions
-----------

Types
~~~~~

$${syntax: type}

Parameters and Arguments
~~~~~~~~~~~~~~~~~~~~~~~~

Each parameter may be labeled with a direction.

$${syntax: dir}
$${syntax: param}

Functions and objects may have type parameters.

$${syntax: tparam}

Arguments can be named.

$${syntax: arg}

Expressions
~~~~~~~~~~~

$${syntax: expr}

The following operators are supported:

$${syntax+: unop binop}
